### genieacs-sim Installation on Ubuntu(One Time)
------------------------------------------------------------------
	TR-069 client simulator for GenieACS.

	To install:
	---------------
	git clone https://github.com/zaidka/genieacs-sim.git
    cd genieacs-sim
    npm install

	To use:
	----------------
	$./genieacs-sim --help
	$./genieacs-sim -s 654321 -u "http://192.168.1.8:7547"
